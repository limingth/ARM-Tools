    开发工具类

  • ads1.2.zip ARM Develeper Suite 开发套件

    • 1. 安装时候选择 Full 安装，而不是 typical 安装
    2. Install licence 文件 Licence.dat 在解压之后的 crack 目录下

  • make.exe 项目管理（配合Makefile使用）

    • 辅助工具类

  • regbit.exe 寄存器察看器（可以查看修改bit）
  • terminal.exe 串口工具（可以灵活修改波特率）
  • H-JTAG-V0.9.2.EXE Hjtag工具（可以烧写NandFlash和配合AXD调试）

    1. 选择 S3c2410/S3c2440 + K9F1208
    2. 设置 RAM 地址为 0
    3. init script 脚本不需要
    4. 最后烧写用 program ，选择需要烧写的 bin 文件，点击 program (注意需要确保 hjtag-server 是运行的)
